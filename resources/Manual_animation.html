Animation Maker

An Addendum to Spectra Analyzer for building animations

Guide to Building Animations

1. Selecting the raw data - Raw data file(Matrix):

  • File Type: Use any file containing a single matrix that can be loaded to Spectra Analyzer. Accepted formats include .txt, .csv, and .xlsx.
  • Simply press Open and navigate to the desired file.

2. Loading the fitting results - Fit data file:

  • Requirement: This must be an .xlsx file generated by the "Fit range" button. Such file can be modified for personal purposes, but it must be structured on the same way as the output fitting file from Spectra Analyzer.
  • For best results, it is better to remove any background model columns from the excel file.
  • Automatic Detection: The application can autonomously search for "Fitting" folders and auto-populate the fields if a valid file is found.

3. Specifying Axis Names:

  • The x-axis and y-axis are labeled "Wavelength (nm)" and "Intensity (a.u.)" by default respectively.
  • Ensure to adjust these labels as per your experimental parameters.

4. Curve Management - Curves found:

  • If the fit data file is read successfully (see Step 2), identifiable curves will be listed here.
  • Make sure to select the checkboxes for any curve you wish to include on the animation.

5. Setting Up Animation:

  • Frame Range: Define the starting and ending frames for your animation.
  • Playback Speed: Set the FPS (frames per second) to control the animation speed.

6. Animation Output:

  • Format: All animations are exported as GIF files by default.

7. Initiating Animation Creation:

  • Click "Create" once all settings are configured.

8. Completion Notification:

  • Indicator: After the process has completed, "Finished" will be displayed in the bottom-left corner.
Warning

A progress bar is not currently available. After starting the process, please patiently wait for the animation to be completed. If using the python code version of the app, progress can be monitored in the console.
